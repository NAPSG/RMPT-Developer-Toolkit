Home Button widget

The Home Button widget zooms the map to the initial map extent.

Configure the Home Button widget

To configure the Home Button widget, complete the following steps:

Steps:
  1. Hover over the Home Button widget and click the show or hide button to show or hide this widget in your app.

    The widgets included with a theme are visible by default.

  2. Hover over the widget and click the Configure this widget button to open the configuration properties dialog box.
  3. Optionally change the name text to replace the default name for this widget.
  4. Click OK to close this window.

Use the Home Button widget

The Home Button widget is automatically enabled when the app starts. Clicking the widget resets the map extent to the map's initial extent.
