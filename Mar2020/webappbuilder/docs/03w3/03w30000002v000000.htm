Widget properties

You can use the following general properties in widgets:

Property

Description

id

Unique ID of the widget set in the config.json file or generated by the app container.

icon

Icon URL of the widget.

config

Config object of the widget.

map

Associated map of a 2D widget.

sceneView

Associated scene of a 3D widget.

appConfig

Location of the app's main config.json file.

folderUrl

URL of the widget folder.

state

Current state of the widget, that is, active, opened, or closed.

windowState

Current window state of the widget, that is, normal, maximized, or minimized.

position

Position of the widget. This is important for on-screen widgets.
