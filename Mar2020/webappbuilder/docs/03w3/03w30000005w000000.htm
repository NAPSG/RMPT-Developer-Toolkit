Swipe

The Swipe widget provides a tool that shows a portion of a layer in a map when you swipe.

The following table lists the configurable attributes of the Swipe widget:

Attribute

Description

style

String. Style of swipe too. The default value is vertical. Available options are vertical, horizontal, and scope.

layerId

String. The ID of the layer to swipe. There is no default value.

Example:

{
    "style": "vertical",
    "layerId": "SuperTuesdaySample_1433"
}
