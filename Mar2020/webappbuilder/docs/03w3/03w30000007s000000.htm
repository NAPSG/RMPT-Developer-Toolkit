Navigate widget

Use the Navigate widget to rotate and pan the scene.

Configure the Navigate widget

There is no configuration for the Navigate widget.

Use the Navigate widget

To use the Navigate widget, complete the following steps:

Steps:
  1. Click the Pan button and drag the scene in the direction you want to pan.
  2. Alternatively, right-click to pan the scene if you have a two-button mouse.
  3. Click the Rotate button to return to the default behavior.
