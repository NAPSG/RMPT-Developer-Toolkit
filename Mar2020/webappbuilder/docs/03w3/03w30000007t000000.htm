Layer List widget

The Layer List widget provides a list of 2D or 3D layers in the scene.

Configure the Layer List widget

There is no configuration for the Layer List widget.

This widget can be set to open automatically when an app starts. To enable this feature, click the Open this widget automatically when the app starts button on the widget, which turns it dark green.

Use the Layer List widget

To use the Layer List widget, do the following:

Steps:
  1. Click the Layer List widget to open it.

  2. Check the check boxes to turn layers on and off in the scene.
  3. Optionally hover over the layer and click the Go to full extent button on the right to zoom your scene to the layer's full extent.
