Slides widget

The Slides widget shows different views of your scene. It only reads the slides captured in the web scene. To create slides in Scene Viewer, see Capture slides.

Each slide stores the state of the scene. A state is the camera position (or extent of your scene), environment settings, selected basemap, and status of the layer visibility. If you navigate to a different view in your scene or change layer visibility, you can click this slide to turn your scene to the state you previously saved in the slide.

Configuring the Slides widget

This widget can be set to open automatically when an app starts. To enable this feature, click the Open this widget automatically when the app starts button on the widget, which turns it dark green.

There is no configuration in the Slides widget.

Using the Slides widget

Steps:
  1. Click the Slides widget to open the slides pane at the bottom of the app. If the web scene does not contain any slides, nothing displays in the pane.
  2. Click a slide to change the view of the scene.
  3. Click the Slides widget again or the X icon on the pane to close it.
