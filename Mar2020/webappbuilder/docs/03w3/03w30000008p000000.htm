Full Screen widget

The Full Screen widget enables you to open full-screen mode.

Configure the Full Screen widget

There is no configuration for the Full Screen widget.

Use the Full Screen widget

To use the Full Screen widget, complete these steps:

Steps:
  1. Click the Full Screen button on the scene to open full-screen mode.
  2. Exit full-screen mode by clicking Exit Full Screen (Esc) or pressing Esc on your keyboard.
