Extent Navigate widget

The Extent Navigate widget allows you to navigate the map to its previous or next extent.

Configuring the Extent Navigate widget

Hover over the Extent Navigate widget and click the eye icon to show or hide this widget on the map. The default is hide.

No configuration is needed.

Using the Extent Navigate widget

Click the arrows on the screen to navigate the map to its previous or next extent.
