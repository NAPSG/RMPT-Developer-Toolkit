Full Screen widget

The Full Screen widget enables you to use full-screen mode.

Configure the Full Screen widget

There is no configuration for the Full Screen widget.

Use the Full Screen widget

Steps:
  1. Click the Full Screen button on the scene to open full-screen mode.
  2. Click the button again or press Esc to exit full-screen mode.

Note:
The Full Screen widget is not supported on mobile devices.
