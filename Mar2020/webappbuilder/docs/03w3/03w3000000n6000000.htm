Get started

This section demonstrates how to create a simple demo widget in Web AppBuilder for ArcGIS. You can view what the final widget looks like in the app at http://[your host name: port]/webappbuilder/stemapp/?config=sample-configs/config-demo.json.

If you want to learn how to create a widget with a meaningful function, see Create a ListView widget.

To test your widget in the builder environment, use this URL: http://[your host name: port]/webappbuilder/?id=stemapp.
